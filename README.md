## Ceres en Red

**Ceres en Red** es la plataforma oficial de servicios del Gobierno de la Ciudad de Ceres.
Permite a los vecinos encontrar y contactar profesionales verificados de distintos rubros,
con información clara sobre sus servicios, zonas de cobertura y disponibilidad.

La plataforma está en producción en: `https://ceresenred.ceres.gob.ar`.

### ¿Qué ofrece la plataforma?

- **Búsqueda de servicios y profesionales** por categorías y filtros.
- **Perfiles detallados de profesionales**, con datos de contacto y experiencia.
- **Flujo guiado “Cómo funciona”** que explica de forma simple cómo usar la plataforma.
- **Interfaz moderna y responsiva**, pensada para usarse desde celulares y computadoras.

### Tecnologías principales

- **Framework**: Next.js (App Router).
- **Lenguaje**: TypeScript.
- **Estilos**: Tailwind CSS y componentes reutilizables.
- **Despliegue**: infraestructura basada en Vercel / contenedores.

### Desarrollo local

Requisitos: Node.js y un gestor de paquetes (`npm`, `pnpm` o `yarn`).

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# La app quedará disponible en
# http://localhost:3000
```
